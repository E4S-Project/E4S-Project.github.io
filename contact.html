Get Support

E4S provides a light-weight layer on top of hundreds of open source software packages, driven by the Spack build tool.

For support, use the following options:

  1. Issues with E4S top layer:Support for E4S itself can be obtained by filing an issue at the E4S-Project GitHub site.
  2. Issues with E4S primary packages: The packages listed on the E4S Product Information page have contact and support information at the associated link for each product.
